Scheme database Database of 230 shemes 131 updated schemes (in sheet "filtered"), 13 departments. The dataset contains the scheme ID, scheme name, department ID, department name, and columns with documents as the column name. For documents, the number "1" in the cells indicate that it is a mandatory document to be eligible for a scheme. Documents with numbers "2" indicate that having any one of the documents is sufficient. Documents ith the numbers "3" and "4" follow the same rule as "2".
The ID for each scheme and department can be found in sheet 2 of the excel sheet.		 View schemes database (excel)
Format of letter to household Contains the text of the signed letter from CLAP to be sent to the household, that contains the schemes they are eligible for and relevant contact numbers. template of letter to household
Code to match schemes to individuals and prepare information needed for letter for each household This code looks at the documents each individual has (from the survey: individual dataset) and checks if they have all the documents required to be eligible for a scheme. It checks it for each of the 131 schemes and generates a dataset with all the individual schemes each person can apply for. Download clap_report.RmD
Database for generating letters to households This excel data will be used to create individual letters in word using mail merge. Download excel, english Download with hindi transliteration
Individual-schemes matched The csv file contains the individuals pid and their eligibility for 131 identified schemes. The column names are scheme codes and 0/1 in the cells imply (no/yes) indicating if they are eligible for the scheme or not. Download excel
