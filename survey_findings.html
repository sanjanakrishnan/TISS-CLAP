Method

Once the survey dataset was merged, coded and cleaned, the data was analysed on aspects of: demography, housing, water and toilets, health infrastructure and status, mental health and documents. The base data can be found in the section on survey data aggregation and coding. Steps

  1. In lines 4-28 in clap_report_support.R, the code is used to produce basti wise individual and househld reports and save them in two directories (HH_data and Ind_data)
  2. Following this, code (clap_report.Rmd) was written in RMarkdown to produce html reports containing the main survey findings. To run the markdown code, run lines 29-53 in clap_report_support.R. The generated reports are stored in a directory called reports
  3. In all the analysis, please note the 'n' value (number of respondants) has been mentioned.
  4. This was done for all 26 bastis and M-Ward. The outputs can be found in this link.
  5. Additionally, a seperate code (income.Rmd) was written to provide a summary of the occupation and income trends in M-Ward
